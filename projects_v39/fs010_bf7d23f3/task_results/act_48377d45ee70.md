# act_48377d45ee70 -- Derivation: DC and AC Josephson Effects from Simplified ODEs

**action_kind**: deduction
**sub-question**: (f) -- Assume n_{s,1} approx n_{s,2} and recover DC and AC Josephson effects.

---

## 0. ODEs from Sub-question (e) -- Restated

From the time-dependent Schrodinger equation `i hbar dPsi/dt = H Psi` with
`H = [[eV, K], [K, -eV]]` and `Psi_j = sqrt(n_{s,j}/2) e^{i phi_j}`, separating
real and imaginary parts yields:

```
(1)  dn_{s,1}/dt = (2K/hbar) sqrt(n_{s,1} n_{s,2}) sin(phi_2 - phi_1)
(2)  dn_{s,2}/dt = (2K/hbar) sqrt(n_{s,1} n_{s,2}) sin(phi_1 - phi_2)
     = -dn_{s,1}/dt
(3)  dtheta/dt = 2eV/hbar - (K/hbar)[sqrt(n_{s,2}/n_{s,1}) - sqrt(n_{s,1}/n_{s,2})] cos theta
     where theta = phi_1 - phi_2
```

Equation (2) confirms Cooper-pair density conservation: the pair density
lost by one electrode exactly equals the density gained by the other.

---

## 1. Simplification of dn/dt

**Assumption**: n_{s,1} ~= n_{s,2} ~= n_s/2, where n_s = n_{s,1} + n_{s,2} is
the total superfluid Cooper-pair density.

Under this assumption:

```
sqrt(n_{s,1} n_{s,2}) ~= sqrt((n_s/2)(n_s/2)) = n_s / 2
```

Also, `sin(phi_2 - phi_1) = sin(-theta) = -sin(theta)`.

Substituting into (1):

```
dn_{s,1}/dt = (2K/hbar) * (n_s/2) * (-sin theta)
            = -(K n_s / hbar) sin theta                      [Eq. 1-simplified]
```

**Current-phase relation**: The supercurrent I_s is the electrical current flowing
through the junction. When Cooper pairs tunnel from side 2 to side 1,
dn_{s,1}/dt > 0, and electrical charge flows from side 2 to side 1.
The current is related to the pair tunneling rate by:

```
I_s = (-2e) * A * L * dn_{s,1}/dt = (2e K n_s A L / hbar) sin theta
```

where A is the junction cross-sectional area and L the effective tunneling length.
Defining the **critical current**:

```
I_c = 2e K n_s A L / hbar    [proportional to K n_s / hbar]
```

yields the canonical **Josephson current-phase relation**:

```
I_s = I_c sin theta                [Josephson Eq. 1]
```

The supercurrent is bounded: -I_c <= I_s <= I_c. For a given phase difference
theta, the junction supports a dissipationless DC supercurrent up to the
critical current I_c.

---

## 2. Simplification of dtheta/dt

Under n_{s,1} = n_{s,2} = n_s/2:

```
sqrt(n_{s,2}/n_{s,1}) = sqrt(1) = 1
sqrt(n_{s,1}/n_{s,2}) = sqrt(1) = 1
```

So the bracketed correction term in (3) vanishes identically:

```
sqrt(n_{s,2}/n_{s,1}) - sqrt(n_{s,1}/n_{s,2}) = 1 - 1 = 0
```

Equation (3) simplifies to:

```
dtheta/dt = 2eV/hbar                                      [Josephson Eq. 2]
```

This is the **Josephson voltage-frequency relation**: the rate of change of
the gauge-invariant phase difference is proportional to the DC voltage across
the junction. The factor 2e/hbar reflects the Cooper pair charge.

---

## 3. DC Josephson Effect

**Condition**: V = 0 (zero DC voltage bias across the junction).

Then:
```
dtheta/dt = 0   =>   theta(t) = theta_0 = constant
```

and:
```
I_s = I_c sin(theta_0)
```

**Physical interpretation**:
- A dissipationless DC supercurrent flows through the junction.
- The current magnitude depends on the fixed phase difference theta_0.
- theta_0 can take any constant value, so I_s can be any value between -I_c and +I_c.
- There is **zero voltage drop** despite the current flow -- this is the defining
  feature of superconductivity and is fundamentally different from Ohmic conduction.
- The DC Josephson effect demonstrates that the junction acts as a **phase-tunable
  inductance**: the effective inductance L_J = hbar / (2e I_c cos theta_0).

**Note on naming**: "DC Josephson effect" refers to the presence of a DC
supercurrent at **zero voltage**. The "DC" refers to the nature of the current
(direct), not the voltage.

---

## 4. AC Josephson Effect

**Condition**: V != 0 (constant DC voltage bias).

Integrating Eq. 2:
```
theta(t) = theta_0 + (2eV/hbar) t
```

Defining the **Josephson angular frequency**:
```
omega_J = 2eV / hbar
```

Substituting into Eq. 1:
```
I_s(t) = I_c sin(theta_0 + omega_J t)       [AC Josephson current]
```

**Physical interpretation**:
- Even though the applied voltage is DC, the current oscillates sinusoidally at
  frequency omega_J = 2eV/hbar.
- This is a **quantum-coherent** effect: the phase difference winds at a constant
  rate, and the current oscillates as a result.
- The oscillation frequency is proportional to the DC voltage.

**Time-averaged current**:

Over one period T = 2pi / omega_J:
```
<I_s> = (1/T) integral_0^T I_c sin(theta_0 + omega_J t) dt
```

Substituting u = theta_0 + omega_J t, du = omega_J dt:
```
<I_s> = (1/T) * (I_c/omega_J) * integral_{theta_0}^{theta_0 + 2pi} sin u du
      = (I_c / 2pi) * [-cos u]_{theta_0}^{theta_0 + 2pi}
      = (I_c / 2pi) * [-cos(theta_0 + 2pi) + cos(theta_0)]
      = (I_c / 2pi) * [-cos(theta_0) + cos(theta_0)]
      = 0
```

The time-averaged DC current is **exactly zero**. A pure Josephson element under
DC voltage bias produces only AC current. This is fundamentally different from
an Ohmic conductor, where V != 0 gives a nonzero DC current I = V/R.

**Contrast with Ohmic conduction**:
| Property | Josephson Junction (DC bias) | Ohmic Conductor |
|----------|------------------------------|-----------------|
| Current response | AC: I(t) = I_c sin(omega_J t) | DC: I = V/R |
| Time average | 0 | V/R (nonzero for V != 0) |
| Energy dissipation | None (reactive) | Joule heating I^2 R |
| Mechanism | Quantum-coherent tunneling | Inelastic scattering |

---

## 5. Josephson Frequency in Practical Units

The Josephson frequency:
```
f_J = omega_J / (2 pi) = (2eV) / (2 pi hbar) = (2e/h) * V
```

where `h = 2 pi hbar` is Planck's constant.

Using CODATA 2018 values:
- e = 1.602176634e-19 C (exact)
- h = 6.62607015e-34 J*s (exact)

```
2e/h = (2 * 1.602176634e-19) / (6.62607015e-34)
     = 3.204353268e-19 / 6.62607015e-34
     = 4.835978484...e14 Hz/V
```

Converting to practical units:
```
f_J [MHz] / V [muV] = 4.835978484e14 Hz/V * 1e6 MHz/Hz * 1e-6 V/muV
                     = 4.835978484e14 * 1e6 * 1e-6  MHz/muV
                     = 483.5978484... MHz/muV
                     ~~ 483.6 MHz/muV
```

This is the **Josephson constant** K_J = 2e/h, which has been measured to high
precision and is used in the **Josephson voltage standard** -- the international
standard for the volt since 1990.

**Examples**:
- V = 10 muV => f_J ~~ 4.836 GHz (microwave, X-band)
- V = 1 mV   => f_J ~~ 483.6 GHz (far-infrared)
- The voltage standard uses f_J = 2e/h * V at microwatt voltages, with
  K_{J-90} = 483597.9 GHz/V (1990 convention).

## 6. Naming Clarification

| Term | Meaning |
|------|---------|
| **DC Josephson effect** | A dissipationless **DC supercurrent** flows at **zero applied voltage**. The current magnitude is I = I_c sin theta_0, determined by the constant phase difference. The "DC" refers to the current, not the voltage. |
| **AC Josephson effect** | Under a **DC voltage bias** (V != 0), the current **oscillates** at frequency omega_J = 2eV/hbar: I(t) = I_c sin(theta_0 + omega_J t). The "AC" refers to the oscillatory current, despite the DC voltage. |

This naming can be counterintuitive: the "DC Josephson effect" has DC current at zero voltage, while the "AC Josephson effect" has AC current at DC voltage. The terminology reflects whether the **current** is DC or AC, not the voltage.

## Summary

Starting from the coupled ODEs for n_{s,j} and theta derived from the TDSE for the
STJ Hamiltonian, applying the symmetric-reservoir approximation n_{s,1} ~= n_{s,2}
~= n_s/2 yields:

1. **Current-phase relation**: I_s = I_c sin theta with I_c ~ K n_s / hbar
2. **Voltage-frequency relation**: dtheta/dt = 2eV / hbar
3. **DC Josephson effect** (V=0): theta constant, I_s = I_c sin theta_0 (DC supercurrent)
4. **AC Josephson effect** (V!=0): I_s(t) = I_c sin(theta_0 + omega_J t) with
   omega_J = 2eV/hbar; time-averaged current is zero
5. **Practical frequency**: f_J = (2e/h)V ~ 483.6 MHz/muV
6. **Naming**: DC effect = DC current at zero voltage; AC effect = AC current
   under DC voltage
