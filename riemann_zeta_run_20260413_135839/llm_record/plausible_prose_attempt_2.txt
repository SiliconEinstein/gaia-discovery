Σ_{T < γ_n < 2T} |D(1/2 + iγ_n)|² = (T log T / 2π) · [Σ_n |a_n|²/n + O(T^{-δ})]

S := Σ_{T < γ_n < 2T} |D(1/2 + iγ_n)|²

∫_T^{2T} |D(1/2 + it)|² dt = T · Σ_n |a_n|²/n + O(X²)

Σ_n |D(1/2 + iγ_n)|² ≤ (log T / 2πμ) · ∫_T^{2T} |D(1/2 + it)|² dt + O(T^{1-δ})

This is because each zero contributes to the integral over a disjoint interval of length μ · 2π/log T, and by the mean value property of |D|² in each interval.

**Step 4: Derive the constraint.**

(T log T / 2π) · Σ_n |a_n|²/n ≤ (log T / 2πμ) · T · Σ_n |a_n|²/n + O(T)

This simplifies to:

(T log T / 2π) ≤ (T log T / 2πμ) + O(T)

Dividing by T log T / 2π:

1 ≤ 1/μ + O(1/log T)

If all gaps exceed μ in normalized form, the pair correlation for separations < μ vanishes, contradicting F(α) > 0 for 0 < α < 1.
